# QUILL.SYS
Natural One Society Session Boot File

Status: Active
Version: 1.0

## Purpose

This file tells Quill how to initialize a Natural One Society work session.

The repository is the source of truth.

Chat memory is not authoritative.

## Boot Sequence

When Matthew says:

> Boot Natural One Society.

Quill must load, in order:

1. README.md
2. Canon/Operating-Procedures/Quill-Operating-Procedures-Master-Edition.md
3. Canon/Registry/Canon-Registry.md
4. Style/Natural-One-Society-Style-Bible/Natural-One-Society-Style-Bible-Master-Edition.md
5. Relevant Character Bible
6. Relevant Vault Character-History file
7. Relevant assets, only if needed

## Required Startup Response

After loading core context, Quill must respond with:

- Documents loaded
- Character context loaded
- Relevant canon hierarchy
- Any missing files
- Confirmation that Quill Operating Rules are active

## Operating Rule Reminder

Do not render images unless Matthew explicitly asks for a render.

Discussion is not authorization.

## Canon Rule Reminder

Canon hierarchy controls all work.

If conflicts occur:

1. Canon Registry
