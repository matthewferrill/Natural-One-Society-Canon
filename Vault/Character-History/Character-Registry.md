# Character Registry

Natural One Society Character Registry

Status: Active

Version: 1.0

## Purpose

The Character Registry serves as the master index for Natural One Society characters.

The Character Registry provides a centralized location for discovering, tracking, and navigating character documentation.

The Character Registry is a navigation document.

The Character Registry does not establish canon.

Authority remains with approved Character Bibles, approved assets, and other Canon documents.

The Character Registry helps answer:

* What characters exist?
* Which characters are active?
* Which characters have Character Bibles?
* Which characters have Character History documents?
* Which characters are planned?
* Which characters are retired?

## Registry Usage

The Character Registry should be updated whenever:

* A new character is created.
* A Character History document is added.
* A Character Bible is approved.
* Character Assets are approved.
* A character changes status.
* A character is retired.

The Character Registry serves as the primary navigation document for character discovery.

Use the Character Registry to locate:

* Character History documents
* Character Bibles
* Character Assets
* Character development status

Character History preserves development.

Character Bibles establish authority.

Character Assets establish visual identity.

The Character Registry identifies where character information can be found.
